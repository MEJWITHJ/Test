About Student Resource Hub

The Student Resource Hub is a web application designed to help students find and share educational resources. Our mission is to create a centralized platform for high-quality learning materials across various disciplines.

This project was created as part of the Introduction to Applications Development course at the School of Information Technology.

Features:

  • Browse resources by category
  • View detailed information about each resource
  • Add new resources to the platform
  • Responsive design for all devices
